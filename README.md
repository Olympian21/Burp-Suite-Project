<h1>Qualys Vulnerability Management Home Lab</h1>


<h2>Description</h2>
Installed and deployed Qualys Virtual Scanner Appliance and accessed is through the Qualys Cloud Platform to indentify and remediate vulnerabilites found on a Windows 10 machine.
<br />

<h2>Software & Environments Used </h2>

- <b>VirtualBox</b>
- <b>Windows 10</b>
- <b>VLC (outdated version)</b>
- <b>Firefox (older version)<b/>
- <b>Qualys Cloud Platform</b>
